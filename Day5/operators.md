Operators

Python has different types of operators for different operations. To create a calculator we require arithmetic operators.
Arithmetic operatorr

| Operator | Operator Name   | Example |
|----------|-----------------|---------|
| +        | Addition        | 15 + 7  |
| -        | Subtraction     | 15 - 7  |
| *        | Multiplication  | 15 * 7  |
| **       | Exponential     | 15 ** 7 |
| %        | Modulus         | 15 % 7  |
| //       | Floor Division  | 15 // 7 |
